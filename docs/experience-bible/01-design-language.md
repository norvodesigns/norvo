# Norvo Designs — Experience Bible
## File 01 · The Design Language (Cross-Era System)

> The shared vocabulary every era inherits. Where File 00 defined *what Norvo is*, this file defines *how it looks, types, moves, spaces, and sounds* — the one system that flexes across all five eras without ever breaking. Read File 00 first.

---

## 1.0 — The Governing Idea: One System, Five Temperaments

Norvo's entire visual identity is a **single restrained system** — one font, four colors, one set of named motions, one spatial logic — that is *re-tempered* across five eras. The system never changes; its *temperament* does. A temperament is the combination of how tightly the system is held, how much color is allowed in, how the type is spaced and scaled, how things move, and how light behaves.

The five temperaments, named for fast reference and used by every team:

1. **Documents** — *austere.* Held tightest. Near-monochrome. Still.
2. **Pages** — *composed.* Structured and decorated. First color, first warmth.
3. **Interfaces** — *operational.* Crisp, modular, responsive. Color as signal.
4. **Experiences** — *immersive.* Deep, atmospheric, in motion. Color as light.
5. **Places (The Observatory)** — *harmonious.* All four colors in calm balance. The fullest, most alive, most settled state.

The arc is one of **gradual loosening and gradual coloring**: from tight, cold, and still toward open, warm, and alive — then resolving into balance rather than excess.

## 1.1 — Geist Mono Evolution

**Geist Mono is the only typeface in the entire experience.** It never changes — not between eras, not between headline and body, not between pages. Its monospaced nature is a thesis: equal-width characters read as *engineered, deliberate, structural* — every letter given equal intention. The illusion of "different typography per era" is created entirely by **spacing, hierarchy, scale, weight, rhythm, animation, and layout** — never by swapping fonts.

Below, each era's exact type treatment. These are observable instructions: a designer should be able to set type to match without a second interpretation.

### Era 1 — Documents · "The Typed Artifact"
- **Feel:** A single typed page in a silent archive. Type is the artifact, not the decoration.
- **Scale & hierarchy:** Minimal hierarchy. Body and heading sit close in size — a heading is distinguished by *position and surrounding space*, not by a large jump in scale. One dominant size for reading, one slightly larger for the rare heading. No oversized display type.
- **Spacing:** Generous, even line spacing — airy, like a manuscript with wide leading. Letter spacing is the font's natural monospace default; nothing tightened, nothing tracked out. Long, calm measure (line length) — text sits in a single narrow column with wide margins on both sides.
- **Weight:** A single regular weight only. No bold. Emphasis, where it must exist, is shown by a thin rule or indentation, not by heavier type.
- **Alignment & rhythm:** Strictly left-aligned, ragged right, like a typewriter. A steady vertical rhythm — every line lands on an invisible baseline grid that feels mechanical and exact.
- **Animation:** Type appears as if *being typed or developed* — characters or lines settling in sequence, left to right, top to bottom, at a calm reading cadence (never a frantic typewriter clatter; a measured, deliberate set). Once set, type is perfectly still.

### Era 2 — Pages · "The Composed Column"
- **Feel:** Type that has learned layout. The page is now *designed* — margins, columns, rules, and considered hierarchy.
- **Scale & hierarchy:** A clear three-step hierarchy emerges — a confident heading, a sub-level, and body. The heading grows meaningfully larger than body (a real, visible jump), but still tasteful, never display-huge.
- **Spacing:** Tighter, more intentional line spacing than Era 1 — set like a well-typeset editorial page. Multi-column arrangements appear. Hairline rules separate sections. Letter spacing on small caps and labels is gently tracked *out* to signal "this is a label."
- **Weight:** Two weights now coexist — regular for body, a heavier weight for headings. Used sparingly and structurally.
- **Alignment & rhythm:** Still predominantly left-aligned, but now justified-feeling within disciplined columns and margins. The baseline grid is denser and more editorial.
- **Animation:** Type arrives by *settling into layout* — headings rise a short distance into place and lock; columns fade up in reading order. Movement is graceful and brief, never bouncy.

### Era 3 — Interfaces · "The Labeled System"
- **Feel:** Type as the language of software — labels, values, states, controls. Functional, precise, legible at a glance.
- **Scale & hierarchy:** Hierarchy becomes *modular* — heading, label, value, caption, micro-label. Sizes are tighter and more numerous; type is organized into components and rows. Numerals and data feel native here (monospace's natural strength).
- **Spacing:** Compact and exact. Labels are set small, in uppercase, tracked *out* for clarity. Values sit aligned in columns (monospace makes numbers line up perfectly — used deliberately). Line spacing tightens to interface density without crowding.
- **Weight:** Regular and medium dominate; a heavier weight reserved for the single key value or active label in a control. Weight now signals *state* (active vs. idle).
- **Alignment & rhythm:** Mixed alignment by function — labels left, values right-aligned in their fields. The rhythm is grid-snapped and crisp.
- **Animation:** Type changes *state* rather than merely appearing — values tick or roll when they update (a short mechanical count, not a slot-machine spin), labels brighten when active. Motion is responsive and immediate, tied to interaction.

### Era 4 — Experiences · "The Spatial Word"
- **Feel:** Type that lives in space. Words have depth, parallax, and atmosphere — they exist *in* the scene, not on a flat page.
- **Scale & hierarchy:** Hierarchy becomes *spatial and dramatic.* A few words become large and near; supporting type recedes smaller and farther. Scale contrast is at its boldest here — the largest display moments of the whole journey live in Era 4. Headlines can be commanding.
- **Spacing:** Expressive. Key phrases may be tracked tightly for impact or opened wide for grandeur, chosen per phrase. Type is positioned freely in the composition (off-grid where it serves depth) but always intentionally placed.
- **Weight:** The fullest weight range used — light for atmospheric recede, heavier for foreground impact. Weight conveys distance and importance.
- **Alignment & rhythm:** Freed from the column. Type is placed by spatial composition — centered, offset, layered front-to-back. Still always legible; freedom never costs clarity.
- **Animation:** Type moves *through space* — phrases drift forward as the visitor advances, words parallax at different depths, headlines ease in from slightly behind the picture plane and settle near. Motion is fluid, weighty, and continuous (the Drift and Parallax behaviors, §1.3). Never jittery.

### The Future — Places (The Observatory) · "The Luminous Readout"
- **Feel:** Type as the calm, lit language of a finished place — like the precise, glowing labeling of a fine instrument or an archive's wayfinding. Confident, quiet, and luminous.
- **Scale & hierarchy:** Hierarchy is at its most *complete and harmonious* — every level (from grand title to micro-label) coexists in balance. Nothing shouts; everything is exactly as large as it needs to be. The drama of Era 4 resolves into poise.
- **Spacing:** Open, generous, and exact at once. Labels are tracked out and small, set with the precision of an instrument panel; headings breathe. Spacing feels *designed by someone with nothing left to prove.*
- **Weight:** The full range, used with restraint — light and regular carry most of the experience; medium and heavier mark the few true focal points. Weight now signals *importance within a calm system.*
- **Alignment & rhythm:** A confident, slightly architectural grid returns — but a loose, breathing one, not the tight column of Era 1. Wayfinding labels align to the environment's architecture.
- **Animation:** Type *settles and glows* — it eases into place and holds, with the faintest living shimmer of light across the Signature Gradient on key words (slow, like light moving on metal). Idle type is calm; active type warms. Motion is the most refined of the journey — nothing is hurried, everything arrives.

**The single rule across all five:** the visitor must never consciously notice the font is the same everywhere. They should feel five *worlds*, and only on reflection realize it was one font, evolving.

## 1.2 — The Four-Color Evolution Arc

Four colors. No fifth, ever. They evolve in *prominence and role* across the eras — color **enters gradually** as the web evolves — but none is ever removed. By The Observatory, all four coexist in calm harmony.

**The palette:**
- **Graphite `#14161A`** — structural dark: ground, ink, depth, shadow.
- **Archive White `#F4F5F7`** — soft light: surfaces, paper, clarity.
- **Norvo Violet `#6D5DFB`** — signature energy: signal, intelligence, the spark of the new, interactivity, life.
- **Observatory Gold `#D8B46A`** — warm prestige: value, craft, arrival, the gold of a lit instrument.
- **Signature Gradient** — `linear-gradient(120deg, #6D5DFB, #D8B46A)` — violet flowing into gold; the brand's soul, reserved for the most meaningful moments.

The arc below specifies each color's exact role and prominence per era.

### Era 1 — Documents · *Monochrome*
- **Graphite:** Ink. Carries nearly all the visible content — text and the rare hairline.
- **Archive White:** Ground. The page is Archive White; it dominates the field. The era is essentially a two-tone study of ink on paper.
- **Norvo Violet:** **Absent from view but present as a whisper** — it exists in exactly *one* place: the live, interactive element the visitor must act on (the single forward affordance). A lone violet point in a monochrome world — the first sign of life. Nothing else is violet.
- **Observatory Gold:** Not yet present. (Gold has not entered the story.)
- **Feeling:** Cold, quiet, archival. Color is a held breath.

### Era 2 — Pages · *First Warmth*
- **Graphite:** Still primary ink and now also used for structural rules and section dividers.
- **Archive White:** Still the dominant ground, but now *composed* — framed by margins and columns.
- **Norvo Violet:** **Enters as accent.** Violet now marks links, the active heading, and interactive affordances — the color of "this is alive, you can act here." Used sparingly and precisely; it is a highlight, not a flood.
- **Observatory Gold:** **First appearance — a single touch.** Gold makes its debut as one small mark of value or emphasis (a key initial, a rule under a title, an editorial flourish). It is rare and precious here — the first warmth entering a cool world.
- **Feeling:** The page warms slightly. Color has arrived but is disciplined.

### Era 3 — Interfaces · *Color as Signal*
- **Graphite:** Becomes a true surface color, not only ink — panel grounds and dark interface fields appear. The dark begins to hold space, not just letters.
- **Archive White:** Light surfaces, panels, and high-legibility text fields.
- **Norvo Violet:** **Promoted to the system's signal color.** Violet now means *active, focused, interactive, on.* Active controls, focus states, live values, primary actions — all violet. It is the most functional color in this era and appears throughout, but always *meaningfully* (color = state).
- **Observatory Gold:** **Grows from touch to accent.** Gold now marks *value and achievement* — key metrics, highlighted results, the one number that matters (the impact stats live comfortably here). More present than Era 2, still selective.
- **Feeling:** Crisp, operational, legible. Color carries information.

### Era 4 — Experiences · *Color as Light*
- **Graphite:** Becomes *atmosphere and space* — the deep dark of an immersive environment, the void behind layered depth.
- **Archive White:** Becomes *light itself* — glows, highlights, luminous edges, the lit foreground.
- **Norvo Violet:** **Becomes ambient energy.** Violet now suffuses the atmosphere — glows, light leaks, energy in motion, the color of the immersive world being *alive.* It moves and breathes here.
- **Observatory Gold:** **Rises to a co-lead.** Gold now provides warmth and depth alongside violet — golden light, warm highlights, the contrast that gives the scene richness. Violet and gold begin to *meet.*
- **The Signature Gradient:** **Emerges as a living presence** — violet-to-gold light flows through the most important moments, foreshadowing the harmony to come. This is where the gradient first becomes a *force,* not a flourish.
- **Feeling:** Deep, warm, alive, atmospheric. Color is light and motion.

### The Future — Places (The Observatory) · *Harmony*
- **Graphite:** The calm architecture of the place — deep, settled structure and the dark of the surrounding future.
- **Archive White:** The clean light of the interior — lit surfaces, clear labels, the brightness of a place built for clarity.
- **Norvo Violet:** **A confident, balanced presence** — the living signal of the place: interactive elements, active wayfinding, the pulse of intelligence. Settled, not frantic.
- **Observatory Gold:** **A confident, balanced presence** — the prestige of arrival: warmth, value, the gold of finished craft and lit instruments.
- **The Signature Gradient:** **Fully realized and at peace.** Violet and gold coexist in harmony — neither dominates; together they are the resolved chord the whole journey was climbing toward. The gradient appears at the true focal moments (the brand statement, the primary call to act) in full, calm glory.
- **Feeling:** Balanced, luminous, complete. All four colors live together as one chord.

**The arc in one line:** *Graphite and Archive White begin alone → Violet enters as a single spark → Gold enters as a precious touch → both grow into signal and value → both become light → all four resolve into harmony, crowned by the Signature Gradient.*

## 1.3 — The Motion Vocabulary (Named, Reusable Behaviors)

Motion is described by *feel and easing character* — never by code. These are the named behaviors every team uses by name. When a later file says "the panel performs a **Settle**," it means exactly the behavior defined here. Each has a defined *easing character* (the quality of its acceleration), a *duration feel*, and a *signature*.

### The Eight Core Motions

1. **The Settle** — *Arrival.* An element eases into its final position and stops cleanly, with **no bounce, no overshoot.** Easing character: starts with a little speed, decelerates smoothly to a soft, confident stop. Duration feel: about one calm, unhurried beat. Signature: the feeling of something heavy and well-made coming to rest. Used for nearly all "this has arrived" moments.

2. **The Drift** — *Continuous travel.* Slow, weightless, ongoing movement (parallax layers, ambient float, the forward glide through Era 4). Easing character: linear-to-gentle, never starting or stopping sharply; it *is* already in motion. Duration feel: long and breath-paced. Signature: the unhurried float of something suspended in space.

3. **The Parallax** — *Depth through differential motion.* Nearer things move more, farther things move less, as the visitor advances or tilts the device. Easing character: tied 1:1 to input (scroll, gesture, or device tilt) — it follows the visitor exactly, with the faintest lag on the farthest layers to sell distance. Signature: the world having genuine front-to-back depth.

4. **The Reveal** — *Becoming visible.* Content emerges by *developing into being* — fading up while easing a short distance into place (typically rising slightly, or arriving from a hair behind the picture plane). Easing character: gentle ease-out, never a hard pop. Duration feel: brief, in reading order. Signature: content that *appears as if it was always there, just now lit.*

5. **The Bloom** — *Color and light entering.* The signature behavior of the color arc: monochrome elements *warm into color,* or violet/gold light spreads outward from a point like light filling a space. Easing character: slow, organic, decelerating — light does not snap on, it *grows.* Signature: the moment an era's color "enters." Reserved for color-arrival beats and the Signature Gradient coming alive.

6. **The Shift** — *Era-to-era transformation (the seam).* The gradual morph at every era boundary, where the entire world re-tempers — spacing loosens, color enters, layout reorganizes — *continuously,* never as a cut. Easing character: a long, smooth cross-dissolve of *state,* slow enough to witness, paced to the visitor's forward progress so they feel they are *traveling into* the next era. Signature: evolution you can watch happen. (Each era file specifies its own incoming and outgoing Shift in detail.)

7. **The Warp** — *The Time Warp only.* The single most intense motion in the journey: acceleration through transformation between Experiences and Places. Easing character: a deep accelerate-then-decelerate — the visitor speeds up into a rush of streaking, gradient-lit motion, then *decelerates* to a calm, luminous stop inside The Observatory. Signature: the climactic peak; used exactly once. (Fully specified in File 06.)

8. **The Pulse** — *Living signal.* The faint, slow "breathing" of interactive and live elements — a soft glow that rises and falls, marking that something is alive and actionable. Easing character: a smooth, continuous in-and-out, like calm breathing, never blinking or flashing. Signature: the quiet heartbeat of the interactive layer; the violet spark in Era 1 pulses; the active controls in Era 3 pulse; the call-to-act in The Observatory pulses.

### Motion Principles (govern all eight)
- **No bounce, ever.** Norvo motion is weighty and confident; nothing springs, jiggles, or rubber-bands. Things *settle.*
- **No teleporting.** Nothing appears or vanishes instantly. Everything transitions; every seam is witnessed.
- **Respect for momentum.** Motion follows physical intuition — heavier/nearer things carry more weight; light/far things move freely.
- **Motion serves story or action.** Decorative motion that serves neither is cut.
- **Reduced-motion is a first-class variant.** Where a visitor prefers reduced motion, the Drift/Parallax/Warp soften into gentle cross-fades and the experience stays equally legible and navigable (detailed in File 17). The *meaning* of every motion survives even when its movement is removed.

## 1.4 — Spacing & Grid Philosophy

Spacing is the most powerful storytelling tool after color, and it **loosens across the journey** — the system literally breathes more as the web "evolves."

- **Era 1 (Documents):** A single narrow column, wide symmetrical margins, generous even line spacing. The grid is *implied and strict* — a mechanical baseline rhythm — but only one column exists. Space feels archival and reserved.
- **Era 2 (Pages):** A true multi-column editorial grid appears, with disciplined margins, gutters, and hairline rules. Space is *composed* — deliberately balanced. Slightly tighter line spacing within columns; more structure overall.
- **Era 3 (Interfaces):** A modular grid of panels, rows, and fields. Spacing becomes *systematic and denser* — components snap to a tight grid, with consistent internal padding. Space is organized and efficient.
- **Era 4 (Experiences):** The grid *dissolves into depth.* Elements are placed by spatial composition — front-to-back layers, off-grid focal placement — governed by depth and balance rather than columns. Space opens up dramatically; the composition can be vast.
- **The Observatory (Places):** A loose, breathing, slightly architectural grid returns — but generous and confident, the opposite of Era 1's tightness. Space is *abundant and calm,* like a well-designed building with room to move. Wayfinding aligns to the environment's architecture.

**The arc:** *tight single column → composed multi-column → dense modular → dissolved into depth → abundant architectural calm.* The system starts holding its breath and ends breathing freely.

## 1.5 — Material & Light Philosophy

Norvo surfaces are never flat by accident; their material quality evolves with the eras.

- **Era 1 — Paper.** Surfaces read as matte, lit-flat *paper* — no gloss, no depth, no shadow beyond the faintest. Light is even and ambient, like a reading room. The world is a document on a desk.
- **Era 2 — Printed page.** Still paper-like, but now with the subtle dimensionality of fine print — the faintest sense of ink sitting on stock, the lightest separation between layered elements. Light remains soft and even.
- **Era 3 — Surface and screen.** Materials become *interface surfaces* — clean panels with crisp, controlled edges and subtle elevation (the lightest shadow to separate a panel from its ground). Light is precise and directional, like a well-lit workspace. Violet glows mark active states.
- **Era 4 — Volume and atmosphere.** Surfaces gain real *volume and atmosphere* — depth, soft volumetric light, glows, gentle haze in the deep field. Light becomes a character: violet and golden light sources, luminous edges, the Archive White glow of the foreground against Graphite depth. The world has air in it.
- **The Observatory — Architecture and instrument light.** Materials are *refined architecture* — clean structural surfaces (Graphite and Archive White) lit with the warm precision of fine instruments. Light is calm, intentional, and slightly warm; gold provides the glow of finished craft, violet the glow of the living signal. Surfaces feel *built,* permanent, and luminous — never sci-fi, always architecture.

**Across all eras:** light always has a *source and intention.* It is never flat fluorescence and never garish. Glow is reserved for the *alive* (violet) and the *valuable* (gold); structure (Graphite) and clarity (Archive White) carry the rest. Shadow is always soft; Norvo never uses hard, heavy drop shadows.

## 1.6 — Sound Design Language (Optional Layer)

Sound is an *optional, gentle* enhancement — never required for comprehension, always defaulting to off-or-subtle, always with an obvious, persistent mute. When present, it follows the same restrained, evolving logic as the visuals.

- **Tone:** Spatial, minimal, and warm. Never musical "background tracks," never sci-fi whooshes, never UI beeps. Think *atmosphere and presence* — the quiet ambience of a room, the soft acoustic signature of a space.
- **Evolution across eras:**
  - *Documents:* Near silence — the faint, dry hush of a still archive; the softest tactile tick when the single live element is acted upon.
  - *Pages:* A touch more presence — soft, paper-adjacent textures; gentle, low accents marking section arrivals.
  - *Interfaces:* Subtle, precise feedback — soft, warm confirmations on action (never harsh clicks); a calm operational hum beneath.
  - *Experiences:* Atmospheric and immersive — a low, enveloping ambient bed; soft swells tied to depth and motion; the sense of air and space.
  - *The Time Warp:* The one sonic peak — a building, then resolving, swell that crests as the visitor accelerates and lands as they arrive. Restrained, cinematic, brief.
  - *The Observatory:* A calm, warm, lit-room ambience — quiet, present, settled. Soft, refined confirmations on interaction. The sound of arrival and peace.
- **Rules:** Sound never plays without a clear, lasting way to silence it. Sound never carries information the visuals don't. Sound never startles. The mute state is remembered. The experience is *complete and equal* in total silence.

## 1.7 — How the System Holds Together

If a team is ever unsure whether something belongs, run it against the system:
1. **Is the type Geist Mono, treated per this era's spec?**
2. **Are only the four colors used, in this era's roles and prominence?**
3. **Does every motion map to a named behavior in §1.3?**
4. **Does the spacing match this era's place on the loosening arc?**
5. **Does the material and light match this era's temperament?**
6. **Does it serve depth, motion, or intention — and never spectacle alone?**
7. **Does it keep the Usability Covenant (File 00) intact?**

Seven yeses, or it is not Norvo.
